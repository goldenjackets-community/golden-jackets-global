# Project Conventions

## When to Apply
Always apply when editing any file in this repository.

## Rules
- This is the Global Dashboard — a world map showing all active chapters
- Static HTML/CSS/JS — no build tools, no frameworks
- All JavaScript is vanilla
- This site is for Global Admin (Founder) only — not public-facing
- Each chapter is represented as a pin on the world map
- Chapter data (name, lead, URL, status) must be kept in sync with actual chapter repos
- Default branch is `master`
